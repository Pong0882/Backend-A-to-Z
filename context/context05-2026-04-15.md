## VM 구성

```
192.168.100.1   — Windows 호스트
192.168.100.10  — pong-server  (Spring Boot + MySQL + Promtail, 4GB)
192.168.100.20  — monitoring-server (Prometheus + Grafana + Loki, 1.5GB)
192.168.100.66  — k6-server (부하 테스트 전용, 1GB)
```
