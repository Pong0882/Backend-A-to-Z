# General Notes

pong-to-rich 프로젝트와 직접 연관되지 않는 참고 지식을 정리하는 공간

---

## 이 폴더를 쓰는 경우

- 대화 중 궁금해서 물어봤는데 ROADMAP Phase에 없는 개념
- 프로젝트 구현과 무관하지만 알아두면 좋은 지식
- 특정 Phase에 넣기 애매한 참고 자료

## 이 폴더를 쓰지 않는 경우

- ROADMAP의 Phase에 이미 들어있는 개념 → 해당 `phase-N-xxx/` 폴더에 정리
- 구현하다가 나온 개념 → `pending.md` 에 먼저 메모

## 파일 명명 규칙

- 영어 소문자, 하이픈 구분
- 예: `network-glossary.md`, `linux-commands.md`
